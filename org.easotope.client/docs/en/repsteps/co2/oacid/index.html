Purpose

When acid is used to release CO2 from minerals such as calcite or aragonite, the resulting CO2 has a δ18O that is enriched relative to the original mineral. The amount of enrichment depends on the temperature of the acid used. This step corrects the δ18O value to account for the enrichment.

Algorithm

This correction step uses the acid temperature given by the user during data entry to find the δ18O acid fractionation factor (AFF) for the sample type. It then uses this AFF to calculate a corrected δ18O which represents the expected δ18O of the original mineral using the following formula:

acid_corrected_value = ((1000.0 + original_value) / AFF) - 1000.0

Note that if no acid correction is necessary (for example, when measuring atmospheric gas) the AFF can be set to 1.0 for that sample type and no correction will be made.

This correction step also copies the acid temperature and acid fractionation factor from the replicate database into the table.
