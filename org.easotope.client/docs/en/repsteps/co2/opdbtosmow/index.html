Purpose

Easotope uses the VPDB scale when calculating δ18O. Because some labs might also want results reported in the VSMOW scale, this step converts δ18O from the former scale to the latter.

Algorithm

The conversion is done with the following equation:

δ18O_in_vsmow = (1.0 + δ18O_VPDB_VSMOW / 1000.0) * δ18O_in_vpdb + δ18O_VPDB_VSMOW

The constant δ18O_VPDB_VSMOW is a conversion factor whose value is defined in the constants database. See the configuration screen for more information.
